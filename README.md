# SuperStore TimeSeries using DataScience

A Data Science and Machine Learning project focused on analyzing SuperStore sales data using Time Series forecasting techniques.  
This system helps identify sales trends, forecast future performance, and generate valuable business insights through interactive visualizations.

---

## Features

- 📊 Sales trend analysis
- 📅 Time Series forecasting
- 📈 Interactive visualizations
- 🧹 Data preprocessing & cleaning
- 🤖 Machine learning prediction models
- 📉 Forecast evaluation
- 🌐 Dashboard for business insights

---

## 🛠️ Technologies Used

- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Plotly
- Scikit-learn
- Statsmodels
- Streamlit

## 📈 Dashboard Modules

- Dataset Overview
Displays cleaned dataset and statistical summary.

- Sales Analysis
Shows category-wise and region-wise sales trends.

- Forecasting
Predicts future sales using Time Series forecasting models.

- Model Evaluation
Displays forecasting accuracy and performance metrics.

- Business Insights
Interactive charts for better decision-making.
